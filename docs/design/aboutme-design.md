## 3. 콘텐츠 구조

### 인트로

> "안녕하세요. 사용자 경험과 데이터의 흐름을 함께 고민하는 프론트엔드 개발자 박형우입니다."

### 카드 구성 (6개)

| # | 제목 | 아이콘 | 키워드 | 상세 내용 |
|---|------|--------|--------|-----------|
| 1 | 시작 | 🌱 | 국비교육, HTML/CSS | 코로나 시기 웹 퍼블리싱 국비교육 과정을 통해 HTML, CSS, Photoshop, XD를 배우며 웹 업계에 첫 발을 내디뎠습니다. 처음에는 제가 작성한 코드가 실제 웹사이트로 구현되는 과정이 즐거웠고, 자연스럽게 웹 퍼블리셔로 커리어를 시작하게 되었습니다. |
| 2 | 성장 | 🌿 | WordPress, 그누보드, PHP | 첫 직장에서는 WordPress 기반 홈페이지 제작을 경험하며 웹사이트 운영 구조를 이해할 수 있었고, 이후 마케팅 에이전시에서는 산부인과, 피부과, 여성병원 등 다양한 의료기관 홈페이지를 제작하며 실무 경험을 쌓았습니다. 이 과정에서 그누보드와 PHP를 접하게 되었고, 단순히 화면을 구현하는 것을 넘어 사용자가 입력한 정보가 데이터베이스에 저장되고 관리자 페이지에서 관리되는 구조를 경험하게 되었습니다. |
| 3 | 성과 | 🏆 | 50개+ 프로젝트, 프리랜서 | 실무에서는 홈페이지와 랜딩페이지를 포함하여 50개 이상의 프로젝트를 제작하였으며, 그누보드를 활용한 랜딩페이지 제작부터 상담 신청 폼 구축, 관리자 페이지 연동까지 직접 구현했습니다. 또한 이러한 경험을 바탕으로 프리랜서 프로젝트를 진행하며 실제 고객에게 서비스를 제공한 경험도 가지고 있습니다. |
| 4 | 리더십 | 👥 | 팀 리더, 프로세스 구축 | 홈페이지 제작팀의 리더 역할을 맡아 약 1년 3개월 동안 팀원 3명을 관리하며 업무 분배, 일정 관리, 품질 관리 및 제작 프로세스 구축을 담당했습니다. 당시 명확한 제작 기준이 부족한 환경에서 반응형 제작 기준을 정립하고 프로젝트 진행 프로세스를 개선하여 팀 단위로 안정적으로 프로젝트를 운영할 수 있는 환경을 만들었습니다. |
| 5 | 전환점 | 🔄 | GSAP, JavaScript | 프로젝트를 진행하면서 GSAP를 활용한 인터랙션 개발을 접하게 되었고, 사용자 경험을 더욱 풍부하게 만들기 위해 JavaScript 학습의 필요성을 체감하게 되었습니다. 단순히 정적인 화면을 구현하는 것보다 사용자 행동에 따라 변화하는 인터페이스를 만드는 과정에 큰 매력을 느꼈습니다. |
| 6 | 현재와 목표 | 🚀 | Vue, React, AI | 이후 ERP 솔루션 기업에서 근무하며 Vue와 React 기반 프로젝트를 경험했습니다. 퍼블리셔 역할을 수행하며 프론트엔드 개발자와 협업하는 과정에서 컴포넌트 기반 개발 방식과 데이터 중심의 서비스 구조를 접할 수 있었습니다. 현재는 React를 중심으로 프론트엔드 역량을 확장하고 있으며, AI 도구를 활용한 개발 생산성 향상에도 관심을 가지고 있습니다. 퍼블리싱 경험을 바탕으로 뛰어난 UI 구현 능력과 사용자 경험에 대한 이해를 갖춘 프론트엔드 개발자로 성장하는 것이 저의 목표입니다. |
