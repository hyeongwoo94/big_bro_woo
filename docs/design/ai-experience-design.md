# AI 경험 섹션 설계

> 진행 상황에 따라 이 문서가 업데이트됩니다.

## 1. 개요

| 항목 | 내용 |
|------|------|
| 섹션명 | AI 경험 (AI Experience) |
| 위치 | Portfolio 다음 |
| 레이아웃 | 탭 네비게이션 + 프로젝트 상세 + AI 워크플로우 |
| 인터랙션 | 탭 클릭 시 콘텐츠 전환 |

---

## 2. 기능 목록

| 우선순위 | 기능 | 설명 | 상태 |
|----------|------|------|------|
| **Must** | 탭 네비게이션 | 프로젝트별 탭 전환 | ✅ |
| **Must** | 프로젝트 상세 | 제목, 설명, 기술 스택, AI 도구, 자동화 목록 | ✅ |
| **Must** | AI 워크플로우 | 명령 → 페르소나 역할 → 작업 순서 시각화 | ✅ |
| **Must** | AI 활용 장점 | 2x2 그리드 카드로 AI 사용 이점 표시 | ✅ |
| **Must** | 워크플로우 예시 | "예시 보기" 버튼 → 슬라이드 패널 | ✅ |
| **Should** | 반응형 | PC/Mobile 대응 | ✅ |
| **Should** | TechNote | 기술 설명 버튼 | ✅ |

## 5. 데이터 구조

```ts
interface AIProject {
  id: string;
  title: string;
  description: string;
  techStack: string[];
  aiTools: string[];
  workflow: WorkflowStep[];
  aiBenefits?: AIBenefit[];   // AI 활용 장점
  automations?: string[];      // 자동화 목록
}

interface WorkflowStep {
  id: string;
  role: "command" | "director" | "planning" | "code" | "review";
  label: string;
  description: string;
}

interface AIBenefit {
  title: string;
  description: string;
}

interface WorkflowExample {
  role: WorkflowStep["role"];
  label: string;
  content: string;
}
```
