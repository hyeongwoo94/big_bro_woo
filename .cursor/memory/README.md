# 프로젝트 학습 저장소

이 폴더는 프로젝트가 진행되면서 **축적되는 컨텍스트**를 담는다.  
후반으로 갈수록 AI가 이 내용을 참고해 더 효율적으로 도와준다.

## 파일 구성

| 파일 | 용도 |
|------|------|
| `project-context.md` | 프로젝트 개요, 현재 상태, 진행 상황 |
| `decisions.md` | 중요 결정 기록 (왜 이렇게 했는지) |
| `conventions.md` | 이 프로젝트에서 채택한 컨벤션, 패턴 |
| `learnings.md` | 자주 쓰는 패턴, 헷갈렸던 점, 팁 |

## 활용 방식

- **대화 시작 시**: 저장소가 있으면 AI가 먼저 읽고 컨텍스트를 파악한다.
- **결정·구현 후**: 중요한 내용은 해당 파일에 추가해 둔다.
- **반복 질문/패턴**: learnings에 적어두면 다음엔 바로 참고한다.

## 업데이트 주기

- 프로젝트 킥오프 후 → `project-context.md` 초안
- 중요한 결정 시 → `decisions.md` 추가
- 컨벤션 확정 시 → `conventions.md` 추가
- 유용한 패턴 발견 시 → `learnings.md` 추가
